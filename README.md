# Muktar Bello — Web Developer Portfolio

A modern, responsive portfolio built with **React (Create React App)** and **pure CSS**.  
It showcases my skills, projects, and includes a working contact form connected to a Node.js backend (via **Resend**).

---

## 🚀 Features
- Responsive design with smooth navigation  
- Hero, About, skills, Projects, and Contact sections  
- Live demo links for showcased projects  
- Contact form integrated with backend API  
- Social links in footer (LinkedIn & GitHub)

---

## 🛠️ Tech Stack
**Frontend:** React, JavaScript, HTML5, CSS3  
**Backend:** Node.js, Express, Resend (Email Service)  
**Deployment:** GitHub Pages (Frontend) & Render (Backend)
